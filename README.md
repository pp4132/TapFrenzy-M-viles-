# TapFrenzy-M-viles-
TapFrenzy

Tipo de proyecto: Juego casual para Android
Estado: En desarrollo / prototipo

Descripción
TapFrenzy es un juego casual diseñado para partidas cortas y diversión inmediata. Ideal para momentos de espera, ocio o tiempo libre, permite que los usuarios entren en acción con mecánicas simples y entretenidas.

Mecánica principal: Tocar elementos que aparecen rápidamente en la pantalla.
Partidas cortas: Perfecto para jugar en cualquier momento.
Objetivo: Conseguir la mayor puntuación posible en cada sesión.

Público objetivo
Adolescentes y adultos jóvenes (13-40 años)
Personas que buscan entretenimiento rápido y ligero en momentos de espera:
Transporte público
Filas largas
Tiempo libre en casa o fuera

Problema que resuelve
Convierte minutos de espera en una experiencia divertida y estimulante.
Proporciona entretenimiento rápido, adictivo y accesible, sin curva de aprendizaje.

Modos de juego
1. **Modo Arcade:** partidas rápidas con dificultad creciente. Pierdes al fallar o al agotarse el tiempo.  
2. **Modo Niveles:** niveles predeterminados con dificultad creciente y desafíos específicos. Ideal para quienes buscan progreso y metas claras.
